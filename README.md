# Visualization Tool for Electric Vehicle Charge and Range Analysis

## Project Description
This project analyzes electric vehicle charging and range data using Tableau dashboards. It helps visualize vehicle performance, charging efficiency, driving range, and pricing through interactive dashboards.

## Technologies Used
- Tableau Public
- CSV Dataset
- SQL (SQLite/MySQL)
- Flask (for web integration)
- GitHub

## Dataset
ElectricCarData_Clean.csv

## Features
- EV Brand Analysis
- Driving Range Analysis
- Price Comparison
- Charging Performance
- Interactive Dashboard
- Story Visualization
